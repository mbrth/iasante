Brainstorming

Concept : Appli de suivi alimentaire avec IA pour personnes atteintes de : diabète (Type 1, Type 2, Pré-diabète), hypertension artérielle (HTA(Sirine)), problèmes cardiaques marché niche mais avec une forte demande

Objectif : aider à éviter les aliments problématiques et encourager une alimentation adaptée via gamification (-> méthode jeu, points, niveau, défis…)(ex pour le projet +10pointspar repas enregistré +50 semaine parfaite..) et recommandations IA.

Fonctionnalités principales

1. Suivi alimentaire : enregistrement de repas avec détails nutritionnels, calcul automatique : calories, glucides, protéines, graisses, sucre, sodium, graisses saturées, historique des repas (+alertes en tps réel selon le profil médical)

2. Détection intelligente des risques : diabète : alerte si sucre > 15g ou glucides > 30g par aliment, hypertension : alerte si sodium > 500mg par aliment, cardiaque : alerte si graisses saturées > 5g ou sodium > 500mg

3. Recommandations IA (Premium) : sug perso selon le profil médical, score de santé (0-100), explications des choix, planification de repas sur 7 jours, bdd de repas adaptés

(-> la gammification c’est ce dont le prof nous a parlé pour rendre “addict” à l’appli et nous donner envie d’acheter)
4. Gamification; premium, champion, équilibre parfait), streaks : jours consécutifs de suivi, niveaux : progression basée sur les streaks & classement

5. Profil médical personnalisé : conditions médicales (diabète (avec le type, hta, cardiaque), les médocs que la maladie oblige a prendre, les données corporelles et âge du patient -> ce qui permet une adaptation auto des reco

Différences : Gratuit vs Premium

Version gratuite : 3 repas/jour maximum + alertes basiques pour aliments à éviter + historique limité à 7 jours + badges de “base” + scanner de codes-barres limité (ex: 10 scans/jour) + tab de bord simple - ps de recommandations IA

Version Premium (ex 9.99€/mois ou 79.99€/an) : epas illimités +reco IA illimitées et per +planification de repas sur 7 jours + analyse nutritionnelle avancée avec graph + historique illimité + scanner illimité + badges “exclusifs” (équilibre parfait)+ export PDF des rapports ( pour montrer au medecin son avancé)

maybe a ajouter :  wearables (genre montre connecté), chatbot IA nutritionnel

Stratégie : gratuit pour attirer les utilisateurs, premium pour monétiser les fonctionnalités avancées, gamification pour augmenter l’engagement, limitations gratuites pour inciter à l’upgrade
